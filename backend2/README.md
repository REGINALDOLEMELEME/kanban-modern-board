# backend2 (PHP API)

This folder is the PHP replacement for the old Node backend.

## 1) Configure DB
Edit `backend2/config.php`:
- db_host
- db_name
- db_user
- db_pass

## 2) Import DB
Import `backend2/database_dump.sql` in phpMyAdmin.

## 3) Upload
Upload these folders/files to your hosting web root:
- `frontend/*`
- `backend2/*`

## 4) API URL used by frontend
`frontend/script.js` now uses:
`./backend2/api/index.php?route=`

Example requests generated by frontend:
- `./backend2/api/index.php?route=/board`
- `./backend2/api/index.php?route=/cards/12/move`

## Existing local data export (optional)
If you also want to migrate your current local MySQL data (cards already created), run this on your local machine:

```bash
mysqldump -u YOUR_USER -p kanban_db > kanban_data_full.sql
```

Then import that SQL on hosting.
